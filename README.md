## Тема проєкту
Робота з принципами ООП в С#

## Мета проєкту  
Ознайомитися з принципами ООП на мові програмування С#, навчитися використовувати інкапсуляцію, наслідування та поліморфізм.

## Опис  
- Проєкт реалізовано на C#; містить рішення (`.sln`), вихідні файли (`.cs`) та консольну/командну програму.  
- Структура містить кілька папок:  
  - `Wizards/` — класи чарівників, їх властивості, поведінка;  
  - `Models/`, `Factories/` — моделі та фабрики для створення об’єктів/сутностей;  
  - `Clubs/`, `Duels/` — класи, що відповідають за механіку “клубів/дуелей” чи взаємодій між об’єктами;  
  - `Program.cs`, `DuelingSimulation.csproj`, `DuelingSimulation.sln` — точка входу, проєкт та рішення.  

- Проєкт може слугувати як базовий шаблон для симуляції, ігрової логіки або як демонстрація ООП-підходів.

## Як запустити 
- Склонувати репозиторій
- Відкрити рішення DuelingSimulation.sln у середовищі, що підтримує C# (наприклад, Visual Studio або Rider).
- Зібрати (Build) проєкт.
- Запустити програму — виконуватиметься код із Program.cs.
- Переглянути/змінити логіку, класи, фабрики — за потреби, для розширення функціоналу.

## Структура каталогу
  DuelingSimulation.sln — рішення
  DuelingSimulation.csproj — проєкт
  Program.cs — точка запуску
  /Wizards — класи чарівників
  /Models — моделі даних
  /Factories — фабрики для створення об’єктів
  /Clubs — класи клубів / груп / колекцій
  /Duels — логіка дуелей або взаємодій між об’єктами
